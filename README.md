## ✅ Выполненные задания

### Задание 3.1 - Создание пользователя
- **Маршрут**: `POST /create_user`
- **Модель**: Pydantic модель `UserCreate` с валидацией полей
- **Проверки**: 
  - Email должен быть в правильном формате
  - Возраст (если указан) должен быть положительным целым числом
  - Имя обязательно для заполнения

### Задание 3.2 - Работа с продуктами
- **Маршрут**: `GET /product/{product_id}` - получение продукта по ID
- **Маршрут**: `GET /products/search` - поиск продуктов
- **Параметры поиска**:
  - `keyword` (обязательный) - ключевое слово для поиска
  - `category` (опциональный) - категория товара
  - `limit` (опциональный, по умолчанию 10) - максимальное количество результатов

### Задание 5.1-5.3 - Аутентификация на основе cookie
- **Маршрут**: `POST /login` - вход с установкой защищенной cookie
- **Маршрут**: `GET /profile` - защищенный профиль пользователя
- **Маршрут**: `GET /logout` - выход из системы
- **Особенности**:
  - Динамическое время жизни сессии (5 минут)
  - Автоматическое продление при активности (через 3-5 минут)
  - Криптографическая подпись с использованием библиотеки `itsdangerous`

### Задание 5.4 - Работа с заголовками (базовая версия)
- **Маршрут**: `GET /headers/legacy`
- Извлечение заголовков `User-Agent` и `Accept-Language`
- Валидация обязательных заголовков

### Задание 5.5 - Работа с заголовками (Pydantic модель)
- **Маршрут**: `GET /headers` - возврат заголовков
- **Маршрут**: `GET /info` - дополнительная информация и заголовок `X-Server-Time`
- Использование модели `CommonHeaders` для переиспользования кода

## 🛠 Технологии

- **FastAPI** - современный веб-фреймворк для Python
- **Pydantic** - валидация данных и управление настройками
- **Uvicorn** - ASGI сервер для запуска приложения
- **itsdangerous** - для криптографической подписи cookie
- **python-multipart** - для обработки form-data
